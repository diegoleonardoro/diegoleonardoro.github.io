Summary: Latitude Vs X

The purpose of this project was to analyze wether changes as you get closer to the equator. To accomplish this analyis, I pulled data from the the OpenWeatherMap API to assemble a data set of over 500 cities.
After assembling the dataset, I used Matplotlib to plo various aspects of the weather vs the latitude. Factors that I looked at include: temperatude, cloudiness, wind speed and humidity. This site provides the source data and visualizations created as part of the analysis, as well as explanations and descriptions of any trends and correlations seen.
